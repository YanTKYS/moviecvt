# 動画簡易変換ツール

> **バージョン: v0.3.2（試作版・検証版）**
>
> 本バージョンは試作版です。本番利用の前に動作確認・ライセンス確認を行ってください。

---

## 概要

一般職員が専門的な動画編集ソフトを使わずに、MP4動画の切り出しとサイズ縮小を行うためのWindowsデスクトップアプリです。

庁内共有・メール添付・資料添付に適した動画ファイルを、職員自身が簡単に作成できるようにすることを目的としています。

---

## 対象利用者

庁内の一般職員（動画編集の専門知識は不要）

---

## 利用環境

| 項目 | 内容 |
|------|------|
| OS | Windows 10 (20H2以降推奨) / Windows 11 |
| ネットワーク | 閉域・庁内ネットワーク（インターネット接続不要） |
| 管理者権限 | 不要（一般ユーザー権限で実行可能） |
| .NET | .NET 8 Desktop Runtime（セルフコンテインド配布の場合は不要） |
| WebView2 | Microsoft Edge WebView2 Runtime（Windows 10 20H2以降・Windows 11は標準済み） |

---

## できること

- MP4ファイルを選択・プレビューする
- 動画を再生して開始位置・終了位置を指定する
- 指定した範囲を切り出す
- **高速カット**（再エンコードなし・高速、開始位置が数秒ずれる場合あり）で選択範囲を出力する
- **圧縮変換**（速度優先・画質優先・標準・容量優先）と解像度（元のまま・720p・480p）を選択して出力する
- **動画全体変換**（開始・終了位置指定なしで動画全体を対象にする）
- プレビュー時の音量スライダー・消音ボタン（出力動画の音量は変更しない）
- **変換中の進捗表示**（経過時間・プログレスバー・進捗率）
- **事前変換**（プレビューできないMP4に対し、同意ダイアログ後に変換後ファイルを作成して自動読み込み）
- **動画情報表示**（ffprobe.exe を配置すると長さ・解像度・コーデック等を画面に表示）
- **変換残り時間（ETA）表示**（進捗率5%超になると残り時間の目安を表示）
- **起動時例外ログ**（未処理例外を `logs/startup_error_yyyyMMdd_HHmmss.log` に記録）

## できないこと（対象外）

- MP4以外の動画形式（AVI, MOV, MKS等）の変換
- 複数ファイルの一括変換
- 複数区間の切り出し
- 動画の結合
- 字幕・音声トラックの編集
- フレーム単位の厳密な編集
- 外部サーバへのアップロード

## セットアップ

### 1. アプリの配置

配布パッケージを任意のフォルダに展開します。

```
MovieConverter/
├── MovieConverter.exe         ← アプリ本体
├── Assets/
│   └── player.html
├── bin/
│   └── ffmpeg/
│       ├── ffmpeg.exe         ← 別途入手して配置（必須）
│       ├── avcodec-*.dll      ┐
│       ├── avformat-*.dll     │ 配布元ZIPの DLL 一式を
│       ├── avutil-*.dll       │ ffmpeg.exe と同じフォルダに配置する
│       ├── swresample-*.dll   │ （ffmpeg.exe 単体では動作しない
│       ├── swscale-*.dll      │  ビルドが存在する）
│       └── その他の .dll      ┘
├── docs/
│   ├── ffmpeg_setup.md
│   └── license_notice.md
└── manuals/
    └── user_manual.md
```

> **重要:** `ffmpeg.exe` は単体配置では動作しないビルドがあります。
> 配布元 ZIP に含まれる **DLL 一式**（`avcodec-*.dll`・`avformat-*.dll` 等）を必ず同じフォルダに配置してください。
> 配置後は `bin\ffmpeg\ffmpeg.exe -version` を実行してバージョン情報が表示されることを確認してください。

### 2. ffmpeg.exe の配置

詳細は `docs/ffmpeg_setup.md` を参照してください。

### 3. 起動

`MovieConverter.exe` をダブルクリックで起動します。

## 使い方（概要）

1. **「ファイルを選択...」** でMP4ファイルを選ぶ（またはドラッグ＆ドロップ）
2. **変換範囲**を決める：動画ロード時に開始=0・終了=動画終端が自動設定される。変更すれば指定範囲のみ変換
3. 圧縮設定を選択する（既定値: **しない（高速カット）**）。圧縮変換を選んだ場合は解像度も選択する
4. **「変換実行」** を押して変換する

詳細は `manuals/user_manual.md` を参照してください。

## 出力ファイル

- 保存先: 入力ファイルと同じフォルダ
- ファイル名: `{元のファイル名}_cut_{yyyyMMdd_HHmmss}.mp4`
- 例: `meeting_cut_20260523_153000.mp4`

入力ファイルは上書きされません。

---

## 個人情報の取扱い

- アプリは個人情報をファイルに保存しません
- 外部サーバへの送信を行いません
- 動画ファイルの内容・管理は利用者・利用部署の責任で行ってください

## 対応形式と注意事項

### 対応形式

初期版は **MP4 / AVC(H.264) / AAC** を主対象としています。

### プレビューに関する注意

- 動画プレビューは WebView2（file:// URI 方式）を使用します
- `file://` URI が主方式、失敗時に仮想ホスト（`https://video.local/`）へ自動フォールバックします
- プレビューできない場合は事前変換のダイアログが表示されます
- 解決しない場合は管理者またはDX担当に相談してください

> **v0.3.1 で追加した「代替プレビュー方式」（WPF MediaElement）について:**
> v0.3.1 でこの機能を追加しましたが、`<UseWpf>true</UseWpf>` の影響により起動時クラッシュが発生しました。
> v0.3.2 では起動安定性を優先し、この機能を無効化しています。代替プレビュー方式は将来のバージョンで再検討します。

### その他の注意

- ffmpeg.exe のライセンスについては `docs/license_notice.md` を確認してください
- **高速カット**は再エンコードしないため高速ですが、開始位置が直前のキーフレームに丸められる場合があります
- **圧縮変換**は動画全体を再エンコードするため、長時間動画では変換に時間がかかります
- 長時間動画では、まず高速カットを試し、必要に応じて圧縮変換を使うことを推奨します
- WebView2 Runtime が必要です（Windows 10 20H2以降・Windows 11は標準でインストール済み）

## よくあるトラブル

| 症状 | 確認事項 | 対応 |
|------|----------|------|
| 「ffmpeg.exe が見つかりません」と表示される | `bin\ffmpeg\ffmpeg.exe` が存在するか | `docs/ffmpeg_setup.md` を参照して配置する |
| 変換実行時に終了コード `-1073741515` | DLL が不足している | 配布元 ZIP の DLL 一式を `bin\ffmpeg\` に配置し、`ffmpeg.exe -version` で動作確認 |
| 動画プレビューが表示されない | WebView2 Runtime がインストールされているか | コントロールパネル→プログラムで「Microsoft Edge WebView2 Runtime」を確認 |
| MP4 なのにプレビューできない | ファイル構造の問題の可能性あり | 事前変換ダイアログを試す。または Microsoft Edge に同じファイルをドラッグして再生できるか確認する |
| 圧縮変換に時間がかかる | 長時間動画・高解像度動画は再エンコードに時間がかかる | まず「しない（高速カット）」を試す。解像度を 720p/480p に変更する |
| 高速カットで開始位置がずれる | キーフレーム単位でカットされるため数秒のずれは仕様 | 精度が必要な場合は「標準」等の圧縮変換を使う |
| 変換後のファイルが見つからない | 出力先は入力ファイルと同じフォルダ | エクスプローラーで入力 MP4 と同じフォルダを確認する |
| アプリが起動しない | EXE がブロックされている可能性あり | ファイルのプロパティ→「ブロックの解除」を確認する |

詳細は `manuals/admin_manual.md` の障害対応セクションを参照してください。

## ビルド方法

```bash
cd src/MovieConverter
dotnet restore
dotnet build -c Release
```

**セルフコンテインド単一ファイル発行（推奨）:**

```bash
dotnet publish -c Release -r win-x64 --self-contained true -p:PublishSingleFile=true -o publish/
```

セルフコンテインド発行では、`publish/` に `MovieConverter.exe` が生成されます。
この場合、.NET 8 Desktop Runtime のインストールが不要になります。

> **注意:** ビルドには Windows 環境（またはWindowsDesktop SDK）が必要です。
> Linux 環境では `net8.0-windows` (WinForms) のビルドはできません。
