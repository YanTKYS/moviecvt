## 1. ビルド確認

## 2. 動作・環境確認

- [ ] 閉域（インターネット非接続）環境で正常に起動する

- [ ] WebView2 Runtime がインストールされた環境で動作する
- [ ] .NET 8 Desktop Runtime がインストールされた環境で動作する（またはセルフコンテインド配布）
- [ ] ffmpeg.exe 未配置時に適切なメッセージが表示される
- [ ] `bin\ffmpeg\` に ffmpeg.exe および DLL 一式（`avcodec-*.dll`、`avformat-*.dll` 等）が配置されている
- [ ] `bin\ffmpeg\ffmpeg.exe -version` を実行してバージョン情報が表示される
- [ ] ffmpeg.exe 配置後に変換が正常に動作する

- [ ] 変換実行でMP4が出力される
- [ ] キャンセルで変換を中断できる
