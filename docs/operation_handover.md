## 1. ツール概要

一般職員がMP4動画の切り出し・サイズ縮小を行うためのWindowsデスクトップアプリ。
専門的な動画編集ソフトを使わず、庁内共有・資料添付に適した動画を簡単に作成できる。

---

## 2. 通常運用手順

### 2-1. 利用者の操作手順（概要）

1. アプリを起動する
2. 「ファイルを選択...」でMP4ファイルを選択、またはドラッグ＆ドロップする
3. 動画を再生して開始・終了位置を確認する
4. 「現在位置を開始に設定」「現在位置を終了に設定」で範囲を設定する
5. 圧縮設定・解像度を選択する
6. 「変換実行」を押す
7. 完了メッセージが表示されたら、出力ファイルを確認する

### 2-2. 出力ファイルの場所

出力ファイルは入力ファイルと同じフォルダに作成されます。

ファイル名形式:
```
{元のファイル名}_cut_{yyyyMMdd_HHmmss}.mp4
例: meeting_cut_20260523_153000.mp4
```

## 3. 定期作業

| 作業 | 頻度 | 担当 | 内容 |
|------|------|------|------|
| FFmpegバージョン確認 | 半年に1回程度 | DX担当 | 最新版への更新要否を判断する |
| WebView2 Runtime の更新確認 | Windows Update に依存 | IT担当 | 通常はWindowsUpdateで自動更新される |
| .NET 8 Runtime の更新確認 | 半年に1回程度 | DX担当 | セルフコンテインド配布の場合は再ビルドが必要 |

---

## 4. 更新作業

### 4-1. アプリ本体の更新

1. 新バージョンのソースコードをビルドする
2. 旧バージョンの実行ファイルをバックアップする
3. 新バージョンの実行ファイルを配置する
4. `bin\ffmpeg\ffmpeg.exe` は既存のものを引き継ぐ（必要な場合は更新）
5. 更新後の動作確認を行う（`docs/test_scenarios.md` を参照）

### 4-2. FFmpegの更新

1. 新バージョンの FFmpeg for Windows を入手する（`docs/ffmpeg_setup.md` を参照）
2. `bin\ffmpeg\ffmpeg.exe` を新バージョンに置き換える
3. 変換テストを行い正常動作を確認する

---

## 5. ロールバック手順

1. バックアップした旧バージョンの実行ファイルを元の場所に戻す
2. 動作確認を行う

---

## 6. 障害時の確認事項

### 6-1. アプリが起動しない

| 確認事項 | 対応 |
|----------|------|
| .NET 8 Desktop Runtime がインストールされているか | Microsoft 公式からインストール（セルフコンテインド配布の場合は不要） |
| アプリファイルが破損していないか | 配布物を再取得して上書きする |
| Windows のイベントログにエラーがあるか | イベントビューアで確認する |

### 6-2. 動画プレビューが表示されない

| 確認事項 | 対応 |
|----------|------|
| WebView2 Runtime がインストールされているか | IT担当に相談してインストールを依頼する |
| `Assets\player.html` が存在するか | 配布物に含まれているか確認する |
| アプリのログ欄にエラーが表示されているか | エラー内容をDX担当に連絡する |

### 6-3. 変換ボタンが押せない・変換できない

| 確認事項 | 対応 |
|----------|------|
| `bin\ffmpeg\ffmpeg.exe` が存在するか | `docs/ffmpeg_setup.md` に従って配置する |
| 開始位置・終了位置が設定されているか | 利用者に手順を案内する |
| 開始位置 < 終了位置 になっているか | 利用者に手順を案内する |

### 6-4. 変換は完了したが出力ファイルがない

| 確認事項 | 対応 |
|----------|------|
| 入力ファイルと同じフォルダを確認したか | ファイル名のパターンで検索する |
| ログ欄にエラーが表示されているか | エラー内容をDX担当に報告する |
| 出力先フォルダへの書き込み権限があるか | IT担当に権限確認を依頼する |

## 8. 対象外事項の周知

以下は本ツールの対象外です。利用部署に周知してください。

- MP4以外の動画形式
- 複数ファイルの一括変換
- 複数区間の切り出し
- 動画結合・字幕編集
- フレーム単位の厳密な編集
