## 1. 概要

このツールは、一般職員がMP4動画の切り出しとサイズ縮小を行うためのツールです。
本手順書は、利用部署側の担当者が行う一次対応・確認事項を説明します。

---

## 2. 利用開始前の確認

### 2-1. ffmpeg.exe の配置確認

変換機能を使うためには、`ffmpeg.exe` が配置されている必要があります。

確認手順:

1. アプリを起動する
2. 画面下部のログ欄に「⚠ ffmpeg.exe が見つかりません」と表示されていないか確認する
3. 表示されている場合は DX推進担当に連絡する

### 2-2. WebView2 Runtime の確認

アプリ起動後に動画プレビュー欄にエラーが表示される場合は、WebView2 Runtime が必要です。
端末の IT担当に確認・インストールを依頼してください。

---

## 3. 利用者への周知事項

- 対応ファイル形式は **MP4(.mp4) のみ** であること
- 元ファイルは上書きされないこと
- 出力ファイルは元ファイルと同じフォルダに作成されること
- 個人情報が含まれる動画の管理は各部署の規程に従うこと

---

## 4. 一次対応フロー

### 4-1. 利用者から「変換できない」と相談を受けた場合

| 確認事項 | 対応 |
|----------|------|
| 入力ファイルが .mp4 か | MP4以外は非対応を案内する |
| 開始・終了位置が設定されているか | `manuals/user_manual.md` の手順を案内する |
| ステータス欄に「ffmpeg.exe 未配置」と表示されているか | DX推進担当に連絡する |
| ステータス欄に別のエラーが表示されているか | エラー内容をメモしてDX推進担当に連絡する |

### 4-2. 利用者から「動画プレビューが表示されない」と相談を受けた場合

| 確認事項 | 対応 |
|----------|------|
| ログ欄に「WebView2」に関するエラーがあるか | IT担当に「WebView2 Runtimeのインストール」を依頼する |
| エラーがなく読み込み中のまま固まっているか | アプリを再起動して再試行する。解決しない場合はDX推進担当に連絡する |

### 4-3. 出力ファイルが見つからない場合

- 入力ファイルと同じフォルダを確認する
- ファイル名のパターン: `{元ファイル名}_cut_{日付時刻}.mp4`
- 変換がキャンセルまたは失敗した場合、ファイルは作成されない

---

## 5. DX推進担当への連絡時に伝える情報

以下の情報をメモして連絡してください。

- [ ] アプリのバージョン（画面タイトルバーに表示）
- [ ] エラーの内容（画面のログ欄またはエラーダイアログの文章）
- [ ] 発生した操作手順
- [ ] 入力ファイルのパスと拡張子
- [ ] 端末のOS（Windows 10 / Windows 11）

---

## 6. 対象外事項の案内

以下の要望があった場合、現バージョン（v0.1.0）では対応していない旨を案内してください。

| 要望 | 対応 |
|------|------|
| MP4以外の変換 | 対象外。次バージョン以降の要望として記録する |
| 複数ファイルの一括変換 | 対象外 |
| 複数区間の切り出し | 対象外 |
| 動画の結合 | 対象外 |
| 字幕・音声の編集 | 対象外 |

対象外機能の追加要望がある場合は、DX推進担当に要望として報告してください。
